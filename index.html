Hardware

O hardware é um conjunto de componentes físicos que compõem um sistema computacional. São responsáveis por executar as instruções de software e processar os dados de entrada e saída, o que garante um sistema com desempenho adequado., Os componentes do hardware são processadores, memória RAM, placas mãe, disco rígido, fonte, placas de vídeo e rede, teclado, mouse e monitor.


Fonte

Uma fonte de alimentação fornece a energia elétrica que é necessária para alimentar todos os componentes internos do dispositivo como placas mãe, processadores, placas de vídeo, unidades de armazenamento etc. Portanto um tipo de equipamento que deve ser escolhido e manipulado com cuidado, afinal, qualquer equívoco pode resultar em provimento inadequado de eletricidade ou em danos à máquina.
